# Sudoku-Solver-With-Backtracking

Solucionador de sudoku feito em python usando o algoritmo de Backtracking

## Dependencies

- python3

### Run

Para fazer o programa rodar você deve usar o seguinte comando no terminal

`~ $ python3 sudoku.py`
